# AI-Driven Customer Churn & Revenue Impact Simulator

This project predicts customer churn using machine learning and visualizes business impact using Power BI.

## Tools Used
- Python
- XGBoost
- Power BI
- Machine Learning

## What this project does
- Predicts churn probability
- Calculates revenue at risk
- Identifies high-risk customers
- Simulates discount strategies using What-If analysis

## Dashboard
The Power BI dashboard shows KPIs and allows decision-makers to test retention strategies.
